## 07_設定 模組 - 測試案例

本文件針對以下四個設定項目與其關聯邏輯，依 `qa-test-case-writer_v14` 規則產出原子級測試案例：

- 公開名片資訊
- 自動同意收入申請
- 自動傳送名片
- 預設招呼語
- 設定之間的關聯情境（雙方設定組合矩陣）

所有案例以「一列只驗一件事」為原則，不合併情境。

| 網路逾時 | 顯示「請稍後再試」或等同文案，不重複寫入資料 |
| 連續點擊 / 重複提交 | 僅執行一次，後續點擊不產生額外請求或重複資料，按鈕可暫時 disabled 或 debounce |
| 網路中斷（操作中途） | 顯示網路錯誤提示，已成功寫入的資料不重複寫入，未完成的操作不視為成功 |
| 多人同時操作同一筆資料 | 以最後成功寫入者為準，避免資料不一致或錯誤狀態；如有衝突應有明確錯誤提示 |
| 流程中途離開再回來 | 回到流程起點或上次停留頁面（依產品設計）；不得殘留錯誤狀態或半成品被誤當作已儲存資料 |
